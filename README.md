### hypixel-api-reborn but CLI

## Installation
`npm i https://github.com/hypixel-api-reborn/cli -g`

## Available commands
`player` - Allows you to get statistics of player

## Usage
`har [command]`

## Options
- `-k` - API key (required)
- `-r` - Raw data
- `-g` - Request for player's guild (command: `player`)

## Example
`player` - `har player StavZDev -k key -r -g`
